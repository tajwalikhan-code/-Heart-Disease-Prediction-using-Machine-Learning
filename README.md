# 📖 Overview

Heart disease remains one of the leading causes of death worldwide.

Early prediction can help healthcare professionals identify patients who may require further medical evaluation.

This project presents a complete **Machine Learning pipeline** that predicts the likelihood of heart disease using **13 clinical health indicators**.

Instead of focusing only on model accuracy, this project demonstrates the **complete AI workflow**, including:

* Data preprocessing
* Exploratory Data Analysis
* Feature Engineering
* Model Training
* Model Comparison
* Performance Evaluation
* Deployment using Gradio

The final result is an interactive web application capable of making predictions in real time.

# 📊 Dataset

The model uses a structured medical dataset containing **13 clinical attributes**.

| Feature             | Description                 |
| ------------------- | --------------------------- |
| Age                 | Patient Age                 |
| Sex                 | Gender                      |
| Chest Pain          | Chest Pain Type             |
| Resting BP          | Resting Blood Pressure      |
| Cholesterol         | Serum Cholesterol           |
| Fasting Blood Sugar | Blood Sugar Level           |
| Rest ECG            | ECG Results                 |
| Max Heart Rate      | Maximum Heart Rate          |
| Exercise Angina     | Exercise-induced Chest Pain |
| Oldpeak             | ST Depression               |
| Slope               | Slope of ST Segment         |
| Major Vessels       | Number of Major Vessels     |
| Thal                | Thalassemia                 |

---

# 🤖 Machine Learning Models

Multiple algorithms were trained and evaluated.

| Model                  | Purpose                   |
| ---------------------- | ------------------------- |
| Logistic Regression    | Baseline Classification   |
| Random Forest          | Ensemble Learning         |
| Support Vector Machine | Non-linear Classification |
| XGBoost                | Best Performing Model     |

The system automatically selects the best-performing model based on ROC-AUC.
